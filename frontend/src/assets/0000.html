A Docker image of this web application is also available; If you prefer to spin up your own container of this site and give it a test run, an image can be found on my DockerHub repository,

chinchalinchin/portfolio DockerHUb

To do so, pull the image from DockerHub,

docker pull chinchalinchin/portfolio:latest

And then publish it on the port of your choice

docker run --publish 8080:8080 chinchalinchin/portfolio:latest

After that, you should be able to access this site on localhost:8080 from a browser of your choice.
