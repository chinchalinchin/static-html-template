I make web applications, all kinds of web applications, such as the one you are now browsing. If you don't believe me, you can find the source code for this web application on GitHub,

chinchalinchin/portfolio

Or, if you prefer to spin up your own container of this web application and give it a test run, an image can be found on DockerHub

, chinchalinchin/portfolio

Pull the image from DockerHub,

docker pull chinchalinchin/portfolio:latest

And then publish it on the port of your choice

docker run --publish 8080:8080 chinchalinchin/portfolio:latest
